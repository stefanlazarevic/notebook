# Alert

Компонента за приказивање важне, често временски ограничене поруке.

Ово је компонента коју корисник не може затворити, стога, програмер мора спровести самоуништавање ове компоненте
уколико по дизајну није замишљена да трајно буде приказана на екрану.

Уколико је корисник у обавези да фокусира компоненту како би је затворио, програмер 
треба да користи `AlertDialog` уместо `Alert` компоненте.
