# Listbox

`Listbox` је компонента која приказује елементе као листу при чему крајњи корисник може да изабере
једну или више понуђених опција. За разлику од `Select` компоненте, `Listbox` може садржати и слике.

`Listbox` компонента поседује имплицитну вредност `aria-orientation` атрибута постављену на `vertical`.

https://www.w3.org/TR/wai-aria-practices-1.2/#Listbox
