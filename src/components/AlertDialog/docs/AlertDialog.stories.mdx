# AlertDialog

Ова компонента представља тип дијалога који приказује поруку попут оне која се приказује путем `Alert` компоненте. 
За разлику од `Alert` компоненте, поред поруке `AlertDialog` оставља могућност употребе интерактивних елемената унутар дијалога.

Корисник **треба** да обезбеди идентификатор за `aria-describedby` атрибут који уперује ка садржају, односно поруци која се приказује.
Када је компонента постављена, фокус **треба** да буде постављен на први интерактивни елемент унутар дијалога.

Уколико дијалог функционише као модал, корисник компоненте је дужан да постави вредност атрибута `aria-modal` на `true`.
Када је `aria-modal` атрибут постављен на `true`, навигација путем `Tab` и `Shift` + `Tab` **не сме** излазити изван оквира `AlertDialog` компоненте.
