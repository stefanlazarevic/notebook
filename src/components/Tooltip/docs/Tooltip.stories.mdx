### Стилизовање

У наставку можете видети CSS за руковање позицијама појашњења који се користи за све
дате примере у наставку документације.

```css
.TooltipContainer {
    position: relative;
}

.TooltipContainer > .Tooltip[data-position="right"],
.TooltipContainer > .Tooltip[data-position="left"],
.TooltipContainer > .Tooltip[data-position="top"],
.TooltipContainer > .Tooltip[data-position="bottom"]
 {
    display: none;

    white-space: nowrap;
}

.TooltipContainer > .Tooltip[data-position="right"],
.TooltipContainer > .Tooltip[data-position="left"]
 {
    top: 50%;

    transform: translateY(-50%);
}

.TooltipContainer > .Tooltip[data-position="top"],
.TooltipContainer > .Tooltip[data-position="bottom"]
 {
    left: 50%;

    transform: translateX(-50%);
}

.TooltipContainer > .Tooltip[data-position="right"] {
    left: calc(100% + 10px);
}

.TooltipContainer > .Tooltip[data-position="left"] {
    right: calc(100% + 10px);
}

.TooltipContainer > .Tooltip[data-position="top"] {
    bottom: calc(100% + 10px);
}

.TooltipContainer > .Tooltip[data-position="bottom"] {
    top: calc(100% + 10px);
}

.TooltipContainer:hover > .Tooltip[data-position="right"],
.TooltipContainer:hover > .Tooltip[data-position="left"],
.TooltipContainer:hover > .Tooltip[data-position="top"],
.TooltipContainer:hover > .Tooltip[data-position="bottom"] {
    display: inline-block;
}
```

### Наговештавање

```tsx
function HintExample() {
	const [showHint, setShowHint] = useState(false);

	function onClick() {
		setShowHint(!showHint);
	}

	return (
		<HelpButton id="help-button" size={20} onClick={onClick} className="HintExample" aria-controls="tooltip-help">
			{showHint && (
				<Tooltip id="tooltip-help" position="right" aria-describedby="help-button">
					You must be over 18.
				</Tooltip>
			)}
		</HelpButton>
	);
}
```
