# AccordionPanel

`AccordionPanel` уз `AccordionHeader` представља градивну компоненту сваке `Accordion` компоненте.

Корисник ове компоненте **мора** обезбедити да `aria-labelledby` атрибут има вредност идентификатора
`AccordionHeader` компоненте која је контролише.
