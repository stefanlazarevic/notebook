# Tabs

Компонента која служи за обележавање група информација под заједничким називом.
Табови могу да се користе за обележавање тренутне интернет странице, тренутног фолдера
у систему фајлова итд.

## Повратни позиви

Функције које се позивају приликом извршења одређених акција над компонентом.

### onSelect

Функција која се позива приликом одабира активног таба.

<code>(event: React.SyntheticEvent, tabIndex: number) => void</code>

### onClose

Функција која се позива приликом затварања таба.

<code>(event: React.SyntheticEvent, tabIndex: number) => void</code>
