# Checkbox

Поље за потврду представља компоненту форме која се користи за одабир вишеструких вредности.
За разлику од прекидача, стање поља за потврду се дефинише једном од три вредности а то су:
`true`, `false` и `"mixed"`. "Мixed" вредност служи за представљање недефинисаног стања.

			checked={select('checked', [true, false, "mixed"], false)} 
			disabled={boolean('disabled', false)}
			onChange={action('onChange')} 
		/>

## Радни примери

### Основни пример

```tsx
function BasicCheckbox(props: any) {
	const [checked, setChecked] = useState(false);

	function onChange(event: React.SyntheticEvent, currentState: boolean) {
		setChecked(!checked);
	}

	return <Checkbox checked={checked} onChange={onChange} />;
}
```
